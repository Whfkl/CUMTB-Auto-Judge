# CUMTB-Auto-Judge(TamperMonkey)
基于油猴的矿大（北京）教务系统（2022）一键评教
# 注意事项
此脚本仅供学习交流使用，希望各位同学给出真实的反馈。制作这个脚本的初衷只是为了一键给矿大的各位好老师值得满分好评 φ(>ω<*)
